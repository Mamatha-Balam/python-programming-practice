## 🧠 Key Learnings

### 1. String Concatenation

* Strings can be combined using the `+` operator
* Example: "Hello" + " World" → "Hello World"

### 2. Common Mistake

* Cannot combine string with integer directly
* This results in a TypeError

### 3. String Repetition

* `*` operator is used to repeat strings
* Useful for patterns and formatting

### 4. Length of String

* `len()` function is used to find number of characters
* Includes spaces as well
