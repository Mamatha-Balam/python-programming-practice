## 🧠 Key Learnings

### 1. Order of Execution

* Python executes code line by line (top to bottom)
* Variables must be defined before use

### 2. Common Errors

* NameError → using a variable before defining it
* IndentationError → incorrect spacing before code

### 3. Indentation

* Python is sensitive to spaces
* Unnecessary spaces at the beginning can cause errors

### 4. BODMAS Rule

Python follows operator precedence:

1. Brackets
2. Orders (powers)
3. Division and Multiplication (left to right)
4. Addition and Subtraction (left to right)
