## 🧠 Key Learnings

### 1. What are Nested Conditional Statements?

* A condition inside another condition
* Used to handle complex decision-making

---

### 2. Nested if Statement

* One if block inside another if block
* Inner condition executes only if outer condition is True

---

### 3. Nested if-elif-else

* Multiple conditions inside another condition
* Useful for detailed decision logic

---

## ⚠️ Important Note

* Avoid too many nested conditions
* Can make code difficult to read and maintain
