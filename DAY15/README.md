## 📌 Overview

This repository contains my Day 15 practice focused on loop control statements in Python.

---

## 📖 Topics Covered

* break statement
* continue statement
* pass statement

---

## 🧠 Key Learnings

### 1. break Statement

* Used to terminate the loop immediately
* Stops execution when condition is met

---

### 2. continue Statement

* Skips current iteration
* Moves to next iteration of loop

---

### 3. pass Statement

* Placeholder statement
* Does nothing but avoids errors

---

## 💻 Practice Programs

* Breaking loop at specific condition
* Skipping even numbers
* Using pass in loop
