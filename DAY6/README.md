## 🧠 Key Learnings

### 1. String Indexing

* Used to access individual characters in a string
* Index starts from 0

Example:
"Hello" → H is at index 0

---

### 2. Negative Indexing

* Used to access characters from the end
* Last character → index -1

---

### 3. String Slicing

* Used to extract a portion of a string
* Syntax: string[start:end]
* Includes start index, excludes end index
