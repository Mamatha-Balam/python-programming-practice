## 🧠 Key Learnings

### 1. What is a List?

* A collection of items stored in a single variable
* Defined using square brackets []

---

### 2. Features of Lists

* Mutable (can be modified)
* Can store different data types
* Ordered collection

---

### 3. List Operations

* Concatenation using +
* Repetition using *

---

### 4. Common List Methods

* append() → Add element
* remove() → Remove element
* sort() → Sort elements
