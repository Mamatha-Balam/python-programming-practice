## 🧠 Key Learnings

### 1. What is a While Loop?

* A loop that executes repeatedly as long as a condition is True

---

### 2. How it Works

* Initialize a variable
* Check condition
* Execute code
* Update variable

---

### 3. Important Concept

* Loop must eventually stop
* Otherwise, it leads to an infinite loop
