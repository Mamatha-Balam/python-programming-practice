## 🧠 Key Learnings

### 1. What are Conditional Statements?

* Used to control the flow of a program
* Execute different blocks of code based on conditions

---

### 2. if Statement

* Executes code only when condition is True

---

### 3. if-else Statement

* Executes one block if condition is True
* Executes another block if condition is False

---

### 4. if-elif-else Statement

* Used to check multiple conditions
* Executes the first True condition
