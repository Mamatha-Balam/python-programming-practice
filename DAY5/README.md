## 🧠 Key Learnings

### 1. What is Type Conversion?

* Changing one data type into another
* Useful when working with different types of data

### 2. Converting String to Integer

* `int()` is used to convert numeric strings into integers

### 3. Converting Float to Integer

* `int()` removes the decimal part (does not round)

### 4. Converting Numbers to String

* `str()` is used to convert numbers into strings
