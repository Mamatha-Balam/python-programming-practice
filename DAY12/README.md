## 🧠 Key Learnings

### 1. What is a For Loop?

* Used to iterate over a sequence (list, string, range, etc.)
* Executes code for each item

---

### 2. range() Function

* Generates a sequence of numbers
* Syntax: range(start, end)
* End value is excluded

---

### 3. Advantages of For Loop

* Cleaner and easier than while loop
* Useful when number of iterations is known
